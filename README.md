# Book Notes

Chapter-by-chapter notes on books I'm reading. One markdown file per book.

These are raw notes for my own learning. Short, opinionated reviews of the same books live on [stephanierebecca.com/books](https://stephanierebecca.com/books/).

## Credit where it's due

The whole idea — the structure, the conventions, even the repo name — is [Michael Parker's](https://github.com/mgp/book-notes). His book-notes repo is where I first saw this format, and it's still the model for everything here. If you want to start your own, his is the better template to read first; mine is just my reading log in the same shape.

## Conventions

- Filenames are kebab-case: `thinking-fast-and-slow.md`
- One H2 (`##`) for the book title; H3 (`###`) for each chapter
- Bulleted notes under each chapter; one self-contained claim per bullet
- No em dashes
- Standard footer attributing the book and linking to the published review

## Source files

PDFs and epubs live in `sources/`, which is gitignored. Drop new books there to read them alongside writing the notes. The repo only tracks the markdown distillations.
